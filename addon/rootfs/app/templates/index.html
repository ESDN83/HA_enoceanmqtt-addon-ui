Actuator Teach-In (Eltako FD62NPN, FSB61, FUD61, etc.)
Before you start: Each sender offset (1-127) can only control one actuator. Use a different offset for each device.
Step-by-Step
  1. Read Base ID — Click "Read" to get the gateway's base address
  2. Enter actuator address — The address printed on the actuator or its packaging (e.g., 0x05834FA4)
  3. Choose sender offset — Pick a unique number (1-127) for this actuator
  4. Put actuator in learn mode:
    • Eltako FD62NPN (Dimmer): Press the rotary knob 4× short + 1× long (>2s) — lamp flickers briefly to confirm learn mode
    • Eltako FSR61 (Switch): Short press the learn button — LED blinks
    • Eltako FSB61 (Blind): Turn the rotary dial to desired run time, then press learn button
    • Other Eltako: Check manual — typically short press the learn/prog button
  5. Click "Send Teach-In" within 30 seconds — the actuator confirms with a brief lamp flicker or LED flash
  6. Add the device with the sender ID shown below using "Manual Entry"
Range important! Place the actuator directly next to the USB300 during teach-in. Some devices (e.g. FD62NPN) have weak radio and won't receive at 2m distance. After successful teach-in, the actuator's receiver typically works at normal range.
Teach-in didn't work?
  • Make sure you use a momentary pushbutton (Taster), not a toggle switch (Schalter)
  • Clear all learned senders: press the learn button 5× quickly (LED flashes = cleared)
  • Try "Repeat Teach-In" — it sends continuously for 30s while you activate learn mode
